# MERN Posts App

A full-stack MERN application that allows users to create, view, and delete posts with image uploads.

## Live Demo

Frontend Live URL

https://mern-posts-frontend-seven.vercel.app/

Backend API

https://mern-posts-backend.onrender.com

Backend Repository

https://github.com/Sriniketh-Vangipuram/mern-posts-backend

---

## Features

* Create posts
* View all posts
* Delete posts
* Upload image thumbnails
* Persistent storage with MongoDB Atlas
* Cloudinary image hosting
* Loading states
* Error handling
* REST API integration
